# 🎮 像素表情包工厂

> 双击 `index.html`，把你的照片变成任天堂风格的像素动图！

## 🚀 怎么用

1. **双击打开** `index.html`（用 Chrome / Edge / Firefox 都行）
2. **上传照片** — 拖进去、点击选、Ctrl+V 粘贴都行
3. **框选人物** — 鼠标拖动矩形框选出你想像素化的人
4. **调整风格** — 拖动滑块选像素大小，切换配色方案
5. **生成动图** — 选动画模板（眨眼/呼吸），下载 GIF！

## 🛠️ 技术说明

- **纯静态网页**，不需要安装任何东西
- 所有处理都在浏览器里完成，不上传任何数据
- 模仿任天堂 NES 红白机的 8-bit 像素风格

## ❓ 常见问题

**Q: 为什么不用联网？**  
A: 所有图片处理都在你电脑的浏览器里完成，不需要服务器。

**Q: 怎么保存做好的 GIF？**  
A: 点"下载"按钮，GIF 会保存到你电脑的下载文件夹。

**Q: 能用手机打开吗？**  
A: 这是电脑版，手机版后续可以考虑。
